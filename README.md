# Intelligent E-commerce Recommendation System

This project combines Data Mining, Graph Analysis, and NLP techniques to build a smart recommendation system for E-commerce platforms. The system analyzes customer purchasing behavior and product reviews to generate intelligent recommendations and insights.

## 🚀 Features

* Frequent itemset mining using Apriori Algorithm and FP-Growth Algorithm
* Product relationship analysis and recommendation generation
* Graph analysis using PageRank Algorithm and HITS Algorithm
* Sentiment analysis on customer reviews using BERT
* Interactive GUI built with [Streamlit](https://streamlit.io?utm_source=chatgpt.com)

## 📂 Datasets

* **Online Retail Dataset** → used for Data Mining, Association Rules, Recommendation System, and Graph Analysis
* **Amazon Reviews Dataset** → used for BERT-based Sentiment Analysis

## 📊 Technologies Used

* Python
* Pandas & NumPy
* NetworkX
* Scikit-learn
* Transformers
* PyTorch
* Streamlit

## 🎯 Project Goals

* Discover products frequently bought together
* Build a recommendation system
* Rank influential products using graph algorithms
* Analyze customer sentiment from reviews
* Improve personalized shopping experience

## 🧠 Algorithms Used

* Apriori Algorithm
* FP-Growth Algorithm
* PageRank Algorithm
* HITS Algorithm
* BERT
